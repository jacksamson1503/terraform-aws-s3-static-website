Source Code
Terraform Files
main.tf
# S3 Bucket
resource "aws_s3_bucket" "my_bucket" {
  bucket = var.bucketname
}

# Ownership Controls
resource "aws_s3_bucket_ownership_controls" "example" {
  bucket = aws_s3_bucket.my_bucket.id
  rule {
    object_ownership = "BucketOwnerPreferred"
  }
}

# Public Access Block — allow public
resource "aws_s3_bucket_public_access_block" "example" {
  bucket                  = aws_s3_bucket.my_bucket.id
  block_public_acls       = false
  block_public_policy     = false
  ignore_public_acls      = false
  restrict_public_buckets = false
}

# ACL — public-read
resource "aws_s3_bucket_acl" "example" {
  depends_on = [
    aws_s3_bucket_ownership_controls.example,
    aws_s3_bucket_public_access_block.example,
  ]
  bucket = aws_s3_bucket.my_bucket.id
  acl    = "public-read"
}

# Upload index.html
resource "aws_s3_bucket_object" "index" {
  bucket       = aws_s3_bucket.my_bucket.id
  key          = "index.html"
  source       = "index.html"
  acl          = "public-read"
  content_type = "text/html"
}

# Website Configuration
resource "aws_s3_bucket_website_configuration" "website" {
  bucket = aws_s3_bucket.my_bucket.id
  index_document {
    suffix = "index.html"
  }
  depends_on = [aws_s3_bucket_acl.example]
}
terraform {
  required_providers {
    aws = {
      source  = "hashicorp/aws"
      version = "6.38.0"
    }
  }
}

provider "aws" {
  region = "us-east-1"
}
variable "bucketname" {
  default = "myterraform-s3-bucket-01042026"
}
Implementation Details
What Was Built
01
S3 Bucket Creation
Created myterraform-s3-bucket-01042026 via Terraform using a variable for the bucket name.
02
Ownership Controls
Set object ownership to BucketOwnerPreferred to allow ACL-based access control.
03
Public Access Enabled
Disabled all public access blocks so the static website is accessible to anyone on the internet.
04
ACL Set to public-read
Applied public-read ACL with depends_on to ensure correct resource creation order.
05
File Upload via Terraform
Uploaded index.html using aws_s3_bucket_object with public-read ACL and correct content type.
06
Static Website Config
Enabled S3 static hosting with index.html as the suffix using aws_s3_bucket_website_configuration.
